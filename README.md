# Calculadora de Partidas Rankeadas

Calculadora de partidas ranqueadas desenvolvida em **JavaScript**, que determina o **nível de um jogador** com base na quantidade de vitórias e derrotas. Ideal para quem quer praticar lógica de programação, funções, operadores, estruturas de decisão e laços de repetição.

## Tecnologias usadas
- JavaScript
- Funções
- Operadores matemáticos
- Estruturas condicionais (if / else)
- Laços de repetição (se futuramente adicionar múltiplos jogadores)
- Entrada e saída no console

## Como executar o projeto?
- clonar o repositório;
- abrir o arquivo script.js no navegador ou rodar no Node.js;
- inserir vitórias e derrotas;
- ler o resultado no console;

## Exemplo de Execução:
   Digite a quantidade de vitórias: 35 | Digite a quantidade de derrotas: 10| O Herói tem de saldo de **25** está no nível de **Prata**

## Regras de Níveis:
| Vitórias | Nível    |
| -------- | -------- |
| < 10     | Ferro    |
| 11 – 20  | Bronze   |
| 21 – 50  | Prata    |
| 51 – 80  | Ouro     |
| 81 – 90  | Diamante |
| 91 – 100 | Lendário |
| ≥ 101    | Imortal  |
